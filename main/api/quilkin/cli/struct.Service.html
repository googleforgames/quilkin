The main entrypoint for listening network servers. When called will spawn any and all enabled services, if successful returning a future that can be await to wait on services to be cancelled.

pub fn spawn_user_space_router( &self, config: Arc<Config>, ) -> Result<(impl Future<Output = Result<()>>, Box<dyn FnOnce(ShutdownRx) + Send>)>

Launches the user space implementation of the packet router using sockets. This implementation uses a pool of buffers and sockets to manage UDP sessions and sockets. On Linux this will use io-uring, where as it will use epoll interfaces on non-Linux platforms.
