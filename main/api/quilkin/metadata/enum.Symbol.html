pub fn resolve_to_bytes<'literal: 'metadata, 'metadata>( &'literal self, metadata: &'metadata DynamicMetadata ) -> Option<Bytes>

Tries to Self::resolve the symbol to a bytes::Bytes, performing a conversion process on different Value if relevant. Returning None if it isn’t supported currently.

  • Value::Bytes The value is copied as-is.
  • Value::String The value is interpreted as a base64 string.
  • Value::Number The value is an eight byte number encoded as big endian.
